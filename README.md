## Project Description
Build a CRUD application in React with Firebase Web SDK v9.  Focus is on implementing the CRUD functionalities, which are:

-Adding a new tast to Firestore
-Getting all tasks from Firestore
-Updating a task
-Deleting a task

## Getting Started

Set up Firestore.  Before we can start using the Firestore service, we need to first create a project and a database on Firebase.

Add a project

Create a database

Set up React.


start our application with this command
```shell
npm start
```
